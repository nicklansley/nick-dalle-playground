The story behind DALL-E is that the engineers wanted to use some AI in reverse! Originally, AI was used to examine an image and use it to create 'alt' text so that vision-impaired people could access the image from this 'alt' description. DALL-E is the result of reversing this AI: Start with the description and use it to generate new images.

So to get started with your prompting, think of way you would describe any particular image using a short sentence!

For example, if you were to describe a picture of a dog asleep in front of a hearth fire, you might say: "a dog asleep in front of a hearth fire". That's it! Describe the image you want to see and get DALL-E to generate it for you!

To get started, look at the prompts and the images generated from them in the Library section of previously created images
